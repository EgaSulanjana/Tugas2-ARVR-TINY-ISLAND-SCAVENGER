# Tugas2-ARVR-TINY-ISLAND-SCAVENGER
Tiny Island Scavenger adalah sebuah pengalaman virtual reality yang imersif dan menenangkan, di mana pemain berperan sebagai seorang kolektor benda alam di sebuah pulau terapung yang mungil. Dengan gaya visual low-poly yang bersih dan ceria, pemain dapat menjelajahi setiap sudut bukit dan lembah pulau untuk menemukan berbagai harta karun alam.
