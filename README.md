# Proyek My First VR Game

Aplikasi virtual reality (VR) sederhana yang dibangun menggunakan Unity Engine untuk mengeksplorasi lingkungan bertema *toon*. Proyek ini difokuskan pada implementasi mekanik dasar VR seperti pergerakan (*locomotion*), interaksi tangan, dan sistem gravitasi pada pemain.

## 🛠️ Detail Teknis

* **Unity Hub**: [Download di sini](https://unity.com/download)
* **Unity Editor Versi**: Unity 6 `2022.3.62f3`
* **Platform**: Android / VR (Meta Quest/Oculus)
* **Fitur**:
* XR Interaction Toolkit Terintegrasi.
* Sistem pergerakan kontinu dengan gravitasi.
* Visualisasi tangan interaktif.


## 📚 Sumber Aset

Proyek ini menggunakan aset eksternal untuk lingkungan yang diperoleh melalui Unity Asset Store:

* **Nama Aset**: [Toon Environments - World Creator Pack Lite](https://assetstore.unity.com/packages/3d/environments/landscapes/toon-environments-world-creator-pack-lite-264325)
